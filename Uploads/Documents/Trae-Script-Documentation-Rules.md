# Trae Script Documentation Rule

## Requirements
Each script needs a `.md` file following `/Users/rd/Library/Mobile Documents/iCloud~md~obsidian/Documents/Obsidian/Scripts/CLAUDE.md`.

### Frontmatter (Required)
```yaml
---
title: "Script Name"
description: "One-line description"
author: riley
category: [Category]
language: [bash|python|etc]
path: "~/Scripts/[Category]/[Sub]/Script Name.ext"
tags: [script, language, category]
---
```

### Sections
1. **Title** (H1)
2. **Overview** (H2)
3. **Features** (H2)
4. **Dependencies** (H2)
5. **Usage** (H2)
6. **Script Details** (H2)
7. **Source Code** (H2) - Exact copy
8. **Examples** (H2)
9. **Notes** (H2)
10. **Related Scripts** (H2)

**Note**: Path in YAML must use `~/Scripts/`.
