# Trae Script Naming Rule

## Conventions
See `/Users/rd/Scripts/.docs/FILENAME_RULES.md`.

### Global Rules
- **Smart Sentence Case**: Capitalize main words. Lowercase articles/prepositions unless first.
- **Acronyms**: Keep uppercase (URL, JSON, API, etc.).
- **Extensions**: Lowercase (.sh, .py, .applescript).
- **Format**: Use **spaces**, avoid kebab/snake case. Name like a menu command.

### Examples
- `Convert JSON to CSV.applescript`
- `Open URL in Browser.applescript`
- `Extract URLs from Text.applescript`
- `Toggle Dark Mode.applescript`
