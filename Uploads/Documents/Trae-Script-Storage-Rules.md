# Trae Script Storage & Quality Rule

## Storage Layout
### `~/Scripts/`
- Organize by Category/Subcategory.
- Keep original filename casing.
- Doc alongside script (`Script Name.md`).

### Obsidian Vault (`.../Obsidian/Scripts/`)
- **Mirror structure** with **Capitalized Spaced** folder names.
- Files: Capitalized Spaced (`DNS Flush.md`).

## Example
`convert-json-to-csv.applescript` (Text/Convert):
1. `~/Scripts/Text/Convert/convert-json-to-csv.applescript`
2. `~/Scripts/Text/Convert/Convert JSON to CSV.md`
3. `.../Obsidian/Scripts/Text/Convert/Convert JSON to CSV.md`

## Checklist
- [ ] YAML correct?
- [ ] Path is `~/Scripts/`?
- [ ] All CLAUDE.md sections?
- [ ] Source code exact?
- [ ] Naming/Category correct?
- [ ] Mirrored to Obsidian?

## Critical
1. Path: `~/Scripts/`
2. Source: Exact copy
3. Structure: Follow CLAUDE.md
4. Naming: Smart Sentence Case
5. Mirror: Capitalized in Obsidian
