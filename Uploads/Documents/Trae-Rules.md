# Trae Script Creation Rule

## Core Rule

**When Trae creates any script to perform a job or task:**

1. **Save the script** in `~/Scripts/` using the existing category/language layout.
   - See [Script_Naming_Rules.md](Script_Naming_Rules.md) for naming.
   - See [Script_Storage_Rules.md](Script_Storage_Rules.md) for storage details.

2. **Create a corresponding `.md` documentation file** alongside the script.
   - See [Script_Documentation_Rules.md](Script_Documentation_Rules.md) for structure.

3. **Create a second copy of the `.md` documentation file** in the iCloud Obsidian vault.
   - Path: `~/Library/Mobile Documents/iCloud~md~obsidian/Documents/Obsidian/Scripts/`
   - Use capitalized folder names and filename.

## Script Location
All scripts created by Trae should be stored in:
`~/Scripts/<category>/<subcategory>/`

**CRITICAL**: Always use `~/Scripts/` NOT `~/.scripts/`

## Category Organization
Follow `/Users/rd/Scripts/.docs/CATEGORY_RULES.md`.
- **Apps/**, **Dev/**, **Languages/**, **Media/**, **Productivity/**, **Riley/**, **System/**, **Text/**, **Utilities/**, **Web/**.
