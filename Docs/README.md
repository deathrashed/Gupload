# Audio Collection

A meticulously organized music library spanning multiple genres, with comprehensive metadata, artwork, and documentation for every release.

## Philosophy

This collection represents years of curation—every file properly tagged, every album accompanied by high-resolution artwork, every artist represented by their logo. It's built on the principle that organization enhances discovery and that metadata matters.

The structure prioritizes:
- **Genre-first organization** for browsing by mood
- **Alphabetical subdivision** for quick navigation
- **Comprehensive artwork** (covers, logos, band photos) at every level
- **Consistent naming** across 10,000+ albums
- **Rich metadata** both in file tags and supplementary markdown docs

## Quick Stats

- **Primary Format**: MP3 (previously FLAC)
- **Base Directory**: `/Volumes/Eksternal/Audio`
- **Top-Level Genres**: 6 major categories
- **Special Collections**: Compilations, splits, singles (genre-specific)
- **Artwork Standard**: album covers + artist logos + supplementary imagery

## What Makes This Different

Most music libraries are chaos. This one isn't. Every album folder contains not just the music, but context—info.txt files, album_info.md with Bandcamp links and track listings, multiple image formats for different use cases. Artists have dedicated documentation files when warranted (see `Death/Death.md` as an example).

Edge cases are handled thoughtfully: compilations live in dedicated `-Compilations-` folders, splits get their own space in genres where they're common (Metal, Punk & Hardcore), and numeric/symbol artist names are cleanly organized in `#` folders.
